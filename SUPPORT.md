# Support

## Usage Questions

For usage questions, setup problems, or general discussion:

- start with `README.md` / `README.ja.md`
- review `CONTRIBUTING.md` if you plan to send a patch
- open a GitHub issue for reproducible bugs or documentation gaps

## Security Issues

For security-sensitive reports, do not use public issues.
Use the private reporting path described in `SECURITY.md`.

## Project Scope Reminder

This project is a trusted-network management UI for Hermes agents.
If you want to run it on a public network, you are responsible for adding an
appropriate authentication and access-control layer in front of it.
